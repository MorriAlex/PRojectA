Выполнял: Мусыркин Алексей - ИСП23В

# О проекте:
 Это десктопное приложение с графическим интерфейсом, демонстрирующее комплексный подход к обеспечению безопасности баз данных и контроля целостности информации. Проект реализует многоуровневую систему защиты, включающую ролевое разграничение доступа, аудит всех операций и механизмы поддержания целостности данных.

# 🔐 Безопасность и разграничение доступа
1. Ролевая модель (admin, manager, employee)
2. Гранулярные привилегии на уровне таблиц и операций (SELECT, INSERT, UPDATE, DELETE)
3. Аутентификация пользователей с хешированием паролей
4. Проверка прав перед выполнением каждой операции

# 📊 Управление данными
1. Интуитивный интерфейс для работы с таблицами
2. CRUD операции с проверкой прав доступа
3. Автоматическая фильтрация данных для рядовых сотрудников
4. Поддержка связанных таблиц и внешних ключей

# 📝 Полный аудит действий
1. Журналирование всех изменений данных
2. История изменений статусов задач
3. Фиксация пользователя, времени и типа операции
4. Хранение как старых, так и новых значений

# 🔄 Контроль целостности
1. Триггеры, предотвращающие некорректные операции
2. Проверка бизнес-правил на уровне БД
3. Каскадное удаление связанных записей
4. Защита от удаления/изменения чужих назначений

# 💾 Резервное копирование
1. Создание резервных копий с временными метками
2. Просмотр списка доступных копий
3. Восстановление базы данных из любой копии
4. Отображение размера файлов копий

# 🧪 Тестирование безопасности
1. Встроенный модуль автоматического тестирования
2. Проверка всех механизмов защиты
3. Демонстрация работы триггеров и ограничений
4. Наглядный вывод результатов
